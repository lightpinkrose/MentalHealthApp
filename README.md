**Mental Health Check-In**

Mental Health Check-In is an Android mobile application designed to help users track their daily mood, stress levels, and personal reflections in a simple, accessible way.

The app focuses on promoting self-awareness and encouraging healthier habits — without overwhelming users with complex features or subscription barriers. It is not intended to diagnose or treat mental health conditions, but rather to serve as a personal tracking tool that users can reference when speaking with healthcare providers.

Core features include secure user registration and login, daily mood tracking, stress level monitoring, optional reminders, and account management.
